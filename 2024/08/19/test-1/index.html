Java中的函数

由于所有的 Java 代码都是类的一部分，我们必须定义函数，使其属于某个类。属于类的函数通常被称为“方法”。

  • 返回类型可类比C++：void,int,double等
  • public前缀表示这个函数是该类的公共方法，相对应的是 private
